# 왕좌의 게임 인터랙티브 시즌 가이드

Game of Thrones 시즌 1~8 완전 가이드 — 시즌별 주요 사건 요약, 등장인물 카드, 인터랙티브 SVG 관계도를 한 화면에서 탐색하는 라이트 모드 웹앱.

## 기능
- S1~S8 시즌 버튼으로 내용 요약 / 등장인물 / 관계도 동시 갱신
- 해당 시즌 등장인물만 카드로 표시 + 시즌별 역할 설명
- SVG 인물 관계도: 가족·동맹·적대·연인·지휘 관계 필터
- 관계도 PNG 저장, 인물 검색, 가문 필터

## 실행
```bash
python3 -m http.server 8080
```
